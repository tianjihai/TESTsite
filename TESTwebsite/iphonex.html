iPhone X 评测：手机的下一个十年


未来手机还未到，但我们看的是手机的未来。

11月3日，iPhone X正式在全球同步上市，iPhone X采用了 和iPhone 8相同的64位的A11 Bonic处理器配置。 在屏幕上，苹果首次在自家手机上使用了OLED面板， 5.8英寸曲面OLED面板分辨率为2436x1165， 支持HDR10以及P3广色域显示。1200万像素双焦段摄像头支持光学防抖， 并具备HDR照片和视频记录功能。在采用全面屏设计后， 苹果应用了FaceID脸部身份识别技术以取代按键TouchID指纹。 iPhone X运行iOS11系统，64GB和256GB存储版本发售。

外观设计

iPhone X 在外观上最叫人注目的，必然是其屏幕顶端的「小刘海」， 点亮屏幕后就会明显看到画面上方有着一个「凹」状缺口。 说实话，虽然这缺口对于操作使用并没有带来影响， 可是不多不少也有影响观感了... 不过这个「小刘海」里藏着的， 却是颠覆苹果通过 iPhone X 展示未来手机应有模样的一个重要部分： TrueDepth 镜头系统。更多有关的功能会在稍后再谈。 虽然就我们肉眼能看到的其实只有扬声器和前置镜头， 但其实里面包含了其他 5 种感应器，所以占用的空间也算是少了（吧？）。 除了 TrueDepth 之外，iPhone X 的正面就只有那片 5.8 吋的 Super Retina HD 屏幕了。 大家习惯的主页键和粗粗边框，在 iPhone X 上面都不复存在， 所以小编在初次点亮屏幕的时候，也稍稍有种被震撼的感觉： 「这台真的是我们常用的 iPhone？」
借测的这台银色版本 iPhone X 的机背是偏银、灰调，配合覆在上面的玻璃背板， 摸上去就有玉石般的温软感觉，非常舒服。苹果方面补充说 iPhone X 的前后两块玻璃都是智能手机界中最耐用的玻璃。 印在上面的就跟 iPhone 8 / 8 Plus 一样， 只有简单的苹果图标和 iPhone 字样， 不过 iPhone X 的双相机模块就改以垂直摆放， 而且更整合了 4-LED True Tone 补光灯到两颗相机之间， 进一步减少开孔，有更具一体化的观感。
那医疗级不锈钢中框上的按键布局虽与 iPhone 7 / 8 系列相似， 但大家惯用的电源键因为在 iPhone X 被赋予了更多的责任， 所以为了让大家更好按到它，这颗侧边按钮的尺吋要稍微比过去变长了一点。 再细看一下，这种经过抛光处理的不锈钢感觉甚至会有种初代 iPhone 的影子呢。 反而太空灰版本的中框是加上了相对低调的黑色亮面 PVD 涂层，也是一个很好的配搭。

屏幕

iPhone X 是苹果首次应用 OLED 材质面板到 iPhone 之上， 官方并大胆表示这是业界中最能抵抗烧屏问题的 OLED 屏幕， 并且是颜色最准确的屏幕（基于 DCI-P3 色域）。 原先小编也有点担心他们会否太习惯于使用 LCD， 而对表现有如野马般难驯的 OLED 没辙。 幸好 iPhone X 的真机表现也没有叫人失望， 在室外也有 625 cd/m2 的最大亮度加持。 配合会按照环境光线而改变显示色调的 True Tone 显示， 这片分辨率为 2,436 x 1,125 的 Super Retina HD 屏幕， 在显示内容时依然细致好看。
iPhone X 有支持 Dolby Vision 和 HDR 10 两种 HDR 格式播放， 所以在打开 Netflix 等的内容就自动对应到有 HDR 显示。 可是，要是你把 iPhone X 和 iPhone 8 Plus 两台同样支持 True Tone 显示的手机屏幕同放一起比较， 会发现 iPhone X 的色调会稍为偏暖一点， 但这已经算是 OLED 屏幕上较容易被接受的效果了。

相信大家都会对画面内容如何迁就 iPhone X 的「小刘海」感到好奇。 如果是原生的播放器，或是 Netflix 等已经有优化的 app， 会首先以较小的画面显示，但有需要的话则可以双击屏幕来填满画面， 这时候「小浏海」就变得非常醒目了。

相机

iPhone X 的双相机组合是两颗同样具备光学防抖的 12MP 相机， 差异就在标准广角镜头的光圈较大，达 f/1.8；长焦距镜头则为 f/2.4， 都能轻易获得足够光量拍摄，低光环境的表现也很不错， 而且标准镜的原生浅景深效果相当明显。 当然，苹果借双相机而做出来的「人像模式」更是因为 A11 Bionic SoC 而获得强化， 拍出来的强行浅景深效果也变得自然一点。不过更重要的还是在 iPhone X 上， 前置相机因为 TrueDepth 镜头系统的关系而补上了， 在拍 Selfie 的时候就能更轻易拍出佳作吧。
相机界面一如往常的简单，提醒一下的就是在 iOS 11 里会预设使用 HEIF 文件格式以 及 HEVC 编码，虽然能把相片及视频所需的存储空间减半， 但要是以电邮或 AirDrop 分享出去的话，一般 app 不一定能打开文件的啊。

Face ID

Face ID 配合 A11 Bionic 处理器的神经网络引擎会一直观察使用者脸部的小变动并作出调整， 所以平常的化妆或剃须都不会干扰 Face ID。 当然，如果脸部真的有相对大的变化的话， Face ID 会要求使用者输入密码去认证。 据指街上随机找个一个陌生人来以 Face ID 成功解锁的机会率只有一百万分之一， 远低于 Touch ID 的五万分之一！ 当然如果你找来一对同卵双生的双胞胎来挑战 Face ID 的话， 这就已经在根本上改变游戏规则了，那倒不如直接偷看密码。
有了面部轮廓的 3D 数据后，Face ID 就可以用作 iPhone X 的安全认证。以往有使用 Touch ID 认证的应用在 iPhone X 上会直接改为使用 Face ID。同样地，面孔的数据都只会存储在 iPhone X 里，并由 A11 Bionic SoC 里的 Secure Enclave 加密保护着，苹果也不能取得你资料啊。相比之下， Face ID 的明显优势在于不需要做一个解锁动作的情况下， 可以直接在锁定页面里解除屏蔽新通知的信息，更为方便。

总结

回应着市场期待的，苹果以 iPhone X 做为纪念 iPhone 改变市场历史的十周年。 在纸面规格上，iPhone X 未必是一台拥有最强大硬件配置的智能手机， 甚至更只是与自家兄弟同级，但却引入了叫人意想不到 的 TrueDepth 镜头系统及几乎是真 · 全面屏的高质 OLED 屏幕。 可是一台手机最重要的还是其系统，在 iPhone X 里 的 iOS 11 是示范了着未来手机该有的姿态 -- 配合无实体键 的全手势操作，会与周遭环境互动的能力（AR）。 当然 iPhone X 并不是来自未来的手机，但它所展示的， 是苹果预言着日后手机的发展路向该是怎么， 所以它至少会是未来手机设计的一个基石。

未来手机还未到，但我们看的是手机的未来。
